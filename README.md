
# Carrito de Compras en Java Web con MySQL

Aplicación web de carrito de compras desarrollada en Java, utilizando JSP y Servlets con una base de datos MySQL. Incluye funciones de gestión de productos, carrito dinámico y procesamiento de pedidos.

![Logo](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/th5xamgrr6se0x5ro4g6.png)


## Autor

- [@theking_eliseo](https://github.com/ELISEOKING)


## Características

- Gestión de Productos
- Carrito Dinámico
- Resumen de Compra
- Interfaz Amigable
